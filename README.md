# Python Intermediário

Bem-vindo ao repositório **Python Intermediário**! Este projeto contém diversos exemplos e aplicações práticas para aprimorar suas habilidades em Python, abordando conceitos intermediários e avançados.

## Estrutura do Projeto

O repositório está organizado em pastas, cada uma representando um projeto ou exemplo prático. Abaixo está uma visão geral:

- **agenda_telefonica/**: Um aplicativo de agenda telefônica com interface gráfica usando `tkinter`.
- **app_tarefas/**: Um gerenciador de tarefas com funcionalidades de adicionar, editar e excluir tarefas.
- **Calculadora_com_python/**: Uma calculadora simples com interface gráfica.
- **calculadora_de_patrimonio/**: Calculadora para calcular patrimônio líquido.
- **calculadora_do_amor/**: Um aplicativo divertido para calcular a "porcentagem de amor" entre duas pessoas.
- **Conversor_de_moedas/**: Um conversor de moedas com integração à API de taxas de câmbio.
- **criando_bot_no_discord_com_python/**: Um bot básico para Discord usando a biblioteca `discord.py`.
- **jogo_de_ligar_lampadas/**: Um jogo interativo para ligar lâmpadas.
- **multiplas_janelas_sem_compilacao/**: Exemplo de navegação entre múltiplas janelas usando `PySimpleGUI`.
- **OOP/**: Exemplos de programação orientada a objetos (POO) em Python.
- **PySimpleGUI/**: Exemplos de interfaces gráficas criadas com `PySimpleGUI`.
- **Scrappy_Site_raspagem_de_dados/**: Um script para raspagem de dados usando `selenium`.
- **seletor_de_cores/**: Um seletor de cores com interface gráfica.
- **tela_de_login_com_tkinter/**: Uma tela de login simples com validação de credenciais.
